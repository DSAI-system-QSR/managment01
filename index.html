📋 구글시트 만드는 방법 보기

📝 구글시트 만드는 방법 (관리자용)

  1. 구글 드라이브 접속: drive.google.com
  2. 새로 만들기 → 구글 스프레드시트 클릭
  3. 아래 "구글시트 템플릿 복사" 버튼을 눌러서 표 복사
  4. 구글시트에 붙여넣기 (Ctrl+V)
  5. 우측 상단 "공유" 버튼 → "링크가 있는 모든 사용자" → "편집자"로 설정
  6. 생성된 링크를 팀원들에게 공유

✅ 완성! 이제 모든 팀원이 같은 시트에서 작업 가능

🎯 전체 프로세스

  1. 관리자: 위 방법으로 구글시트 생성 및 링크 공유
  2. 팀원들: 공유받은 링크로 접속하여 본인 담당 업무 진행률 입력
  3. 카톡 알림: 업데이트 후 "업데이트 완료" 간단 알림
  4. 관리자: 구글시트에서 실시간 전체 현황 확인

💬 카톡 단체방 알림 예시

김철수: 1번 30% 업데이트 완료 (노무사 자문 일정 잡음)

이영희: 3번 60% 업데이트 완료 (절차서 초안 작성 중)

박민수: 4번 85% 업데이트 완료 (분석 완료, 보고서 정리 중)
